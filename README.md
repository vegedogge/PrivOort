This is a privacy-preserving Federated Learning project suitable for IID (Independently and Identically Distributed) data distributions. It utilizes homomorphic encryption to calculate the statistical heterogeneity of clients, enabling the screening of the most appropriate federated learning participants for each round. We have successfully implemented and executed PrivOort across four major open-source frameworks: MindSpore, PaddlePaddle, PyTorch, and TensorFlow.
